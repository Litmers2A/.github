## 프로젝트 소개

<div align="center">
  <img src="https://github.com/user-attachments/assets/0d4de844-0d93-4738-a3a3-ea8b5320dede" />
</div>

<br/>

카카오톡, 전화, 이메일, 노션 등 여러 채널에 분산된 소통과 문서로 인해 PM의 관리 리소스가 낭비되고, 고객사는 프로젝트 진행 상황을 한눈에 파악하기 어렵습니다.

리트머스 파트너스 플랫폼은 프로젝트 진행현황 · 문서 · 회의록 · 해야 할 일 · 문의/요청을 하나의 플랫폼에서 관리하여, 고객사에게는 더 쉬운 확인 경험을, PM에게는 더 효율적인 관리 경험을 제공합니다.

| 문제 | 해결 |
|------|------|
| 분산된 정보와 문서 구조 | 프로젝트 정보와 문서를 한 곳에서 확인할 수 있는 통합 관리 환경 |
| 맥락이 끊기는 커뮤니케이션 | 커뮤니케이션 맥락과 요청 히스토리를 연결해 관리 |
| 흩어지는 후속 작업 관리 | 요청사항과 후속 작업을 체계적으로 관리하는 업무 관리 구조 |

## 기술 스택

### Frontend

| 분류 | 기술 |
|------|------|
| Framework | Next.js 16 (App Router) |
| Language | TypeScript |
| Styling | Tailwind CSS v4 |
| UI Components | shadcn/ui, Radix UI, lucide-react |
| Server State | TanStack Query v5 |
| Client State | Zustand v5 |
| Validation | Zod v4 |

### Backend & Infra

| 분류 | 기술 |
|------|------|
| Runtime | Next.js API Routes |
| ORM | Prisma v7 |
| Database | PostgreSQL (Supabase) |
| File Storage | Supabase Storage |
| Auth | WorkOS AuthKit |
| AI | Google Gemini API, Groq SDK |
| Deployment | Vercel |
| Package Manager | pnpm |
